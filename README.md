🛠️ DigiTools - Premium SaaS Landing Page

### 📝 Description
**DigiTools** is a modern, responsive, and high-performance landing page designed for digital creators and professionals. It features a seamless user interface with a functional product catalog and a dynamic shopping cart experience. Built with a focus on clean code and minimalistic design, this project demonstrates real-time state management and interactive UI components.

---

### 🚀 Technologies Used
* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS (Responsive Design)
* **Icons:** Lucide React & React Icons
* **Notifications:** React-Toastify
* **Deployment:** Vercel / Netlify

---

### ✨ Key Features

* **🛒 Dynamic Shopping Cart:** Users can seamlessly add or remove products from their cart with real-time updates to the cart count in the Navbar.
* **🔄 Interactive View Toggling:** A custom-built logic that allows users to switch between the Product Gallery and the Cart View, providing a focused checkout experience.
* **🔔 Real-time Feedback:** Integrated React-Toastify to provide instant visual feedback for user actions like adding items, removing products, or proceeding to checkout.
